使用方法

1️⃣ 安装依赖

pip install -r requirements.txt

2️⃣ 在 config.py 填写你的 API Key

OPENAI_API_KEY = "你的key"

3️⃣ 运行

python main.py

程序会：

自动爬取新闻
↓
AI分析
↓
生成Excel

生成文件：

data/result.xlsx
